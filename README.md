# Reference
End-to-end DNS encryption with DNS-based ad-blocking. Combines wireguard (DNS VPN), pihole (adblock), and cloudflared (DNS over HTTPS). Created for AWS, GCP, or standalone using Terraform, Ansible, and Docker.
